# 📊 Proyecto de Análisis de Datos: `hr_proyecto_scEmmanuel`

**`hr_proyecto_scEmmanuel`** es un proyecto enfocado en el análisis de datos en el contexto de **Recursos Humanos (HR Analytics)**.  
📚 Desarrollado como parte del curso de **Data Analytics** de la institución **WE Educación Ejecutiva**.

---

## 🧩 Resumen del Proyecto

Este proyecto tiene como objetivo **predecir la promoción de empleados** mediante el uso de técnicas de análisis de datos y modelado estadístico.

---

## 📂 Contenido del Proyecto

### ✅ Caso de Uso

Utiliza un conjunto de datos reales relacionados con empleados, incluyendo:

- Promociones 🌟  
- Puntuaciones de entrenamiento 📈  
- Evaluaciones de desempeño 📋  
- Antigüedad y otras métricas laborales 📊  

📌 **Fuente:** *WNS Analytics Hackathon 2018*

---

### 🧰 Librerías Utilizadas

Este proyecto está construido usando herramientas populares del ecosistema Python:

- `NumPy` 🧮  
- `Pandas` 📊  
- `Scikit-Learn` 🤖  
- `Matplotlib` 📈  
- `Seaborn` 🎨

---

## 🔧 Preprocesamiento de Datos

Se realizó una limpieza y preparación del conjunto de datos incluyendo:

- ✨ Llenado de valores nulos (con moda o mediana)
- 📊 Visualización de valores faltantes
- 🧵 Separación entre variables categóricas y numéricas
- 🧼 Imputación con `SimpleImputer`
- 🔄 Transformación con One-Hot Encoding

---

## 📉 Análisis Descriptivo

Se exploraron los datos mediante:

- Estadísticas de variables numéricas y categóricas
- Distribuciones y visualización de datos 📊

---

## ⚙️ Preparación de Datos para Modelado

- 🧪 División en conjunto de entrenamiento y prueba
- 📐 Escalamiento de variables numéricas con `MinMaxScaler`
- 🚫 Capeo de *outliers* en `length_of_service`

---

## 💾 Exportación

Los datos preprocesados fueron exportados para su uso en futuros modelos o análisis.

---

## 🎯 Objetivo Final

Predecir de forma efectiva la **promoción de empleados** mediante técnicas de análisis y modelado de datos.  
¡Un paso hacia decisiones más inteligentes en el ámbito de Recursos Humanos! 💼
